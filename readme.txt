== Description ==

To improve the customer acquisition, you need to segment your target audience according to their requirements and the budget they may be ready to spend. Then you have to categorise your product into different packages each of which will be appealing to different segments of your potential audience. 

Here we present OS Pricing Tables, an outstanding WordPress Plugin that will help you create product packages easily and quickly. This plugin will help you create beautiful layouts for your product package pages (See the screenshots).

== Features ==

OS Pricing Tables has many mind blowing features that you will love for sure. Have a look at a few important ones.

* Options to add payment gateway button. You can connect to three payment gateways including PayPal, Authorize.net and Firstdata.
* You can add any number of packages.
* You can add any number of features on any packages. 
* 6 beautiful themes to choose
* You can change colors and fonts

== Installation ==

Minimum requirements:    
    
 Wordpress 4.3+    
 PHP 5.4.x    
 MySQL 5.4.x+  

Installation from within backend:

1. In plugin manager, click "Add New".
2. Search for "os-pricing-tables".
3. Click "install".
4. Activate the plugin once it is installed.
5. Click on "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.

FTP upload installation method:
1. Upload the "os-pricing-tables" folder to the "/wp-content/plugins/" directory using your FTP client.
2. Activate the plugin through the "Plugins" menu.
3. Go to the "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the "Plugins" menu.
2. Click "Add New".  
3. Choose the "Upload" option.
4. Click **Choose file** (**Browse**) and select the os-pricing-tables.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.   
7. Go to the "Design Setting" sub menu under WooCommerce menu, finish the installation and manage shop page appearance.
